Titanic Dataset - Data Visualization

📌 Project Overview

This project was completed as part of the CodeAlpha Data Analytics Internship. The objective is to perform data visualization on the Titanic dataset to identify important trends, patterns, and relationships using various charts and graphs.

🎯 Objectives

* Visualize the Titanic dataset using different chart types.
* Understand passenger demographics and survival patterns.
* Compare survival rates based on gender and passenger class.
* Analyze age and fare distributions.
* Present meaningful insights through graphical representation.

📂 Dataset Information

The dataset contains information about Titanic passengers, including:

* Passenger ID
* Age
* Gender
* Passenger Class
* Fare
* Embarkation Port
* Survival Status

🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook / Google Colab

📊 Data Visualizations

The following visualizations were created:

* Survival Count Chart
* Gender Distribution Chart
* Survival by Gender
* Passenger Class Distribution
* Survival by Passenger Class
* Age Distribution Histogram
* Fare Distribution Histogram
* Correlation Heatmap
* Age Distribution by Passenger Class (Box Plot)
* Gender Percentage (Pie Chart)

🔍 Key Insights

* Female passengers had a higher survival rate than male passengers.
* First-class passengers were more likely to survive than third-class passengers.
* Most passengers belonged to the third passenger class.
* The majority of passengers were between 20 and 40 years old.
* Higher fare-paying passengers showed better survival chances.
* Data visualization helped identify important relationships between different variables.
